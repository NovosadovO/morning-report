## Як це працює технічно?

### 1. **Data Collection** (что вже в боте)
- 📊 Вага, BMI, цілі
- 🏃 Біг (Strava), кроки (StepsApp)
- 😴 Сон (QWatch)
- 💰 Крипто (BTC/ETH/AVAX/ONDO, портфель, P&L)
- 📧 Email (кількість, VIP контакти)
- 📅 Календар (события, зміни)
- 🔮 Астро (натальна карта, аспекти, дій)
- 💪 Звички (Armolopid, тренування, медитація)

### 2. **Context Analysis** (Gemini AI)
```python
context = {
    "weight": 83.0,
    "weight_goal": 80.0,
    "weight_trend": "stable",  # или "gaining", "losing"
    "sleep_hours": 4,  # вчора
    "sleep_goal": 8,
    "steps_today": 8420,
    "steps_goal": 10000,
    "run_km": 7,
    "run_goal": 30,  # per week
    "crypto": {
        "btc": {"price": 60164, "change_24h": -7, "portfolio": 0.5},  # in BTC
        "eth": {"price": 1577, "change_24h": -5, "portfolio": 5},  # in ETH
    },
    "emails": {"vip": 1, "important": 3, "spam": 12},
    "calendar": {"events_today": 0, "next_event": "STK 21.06"},
    "astro": {
        "aspects_today": ["☉◻☽ conflict", "♀◇♃ harmony"],
        "energy_level": "mixed",
    },
    "time_of_day": "morning",  # или "lunch", "afternoon", "evening"
}
```

### 3. **Recommendation Generation** (Gemini prompt)
```
Analyze {context} and generate 3 specific recommendations for Oleh.

Rules:
1. Each recommendation must be SPECIFIC (not generic "be healthy")
   ❌ Bad: "Exercize more"
   ✅ Good: "Run 7km at 08:00 because weight is 83kg and goal is 80kg"

2. Each must have WHY (context + data + astro reasoning)

3. Provide EXACT ACTION (time, amount, what to do)

4. Consider astro + health + crypto + habits together

5. Prioritize by impact (health > finance > habits > astro)

6. Language: Ukrainian, warm tone, motivating

Output format:
1️⃣ [PRIORITY] [CATEGORY]
   🎯 ACTION (specific)
   ❓ Чому (reason + numbers + data)
   ⏰ Коли (exact time)
   
2️⃣ ...
3️⃣ ...

✅ SUMMARY + TOMORROW PLAN
```

### 4. **Integration Points**
- `/force_morning` → добавить recommendations блок
- `/force_lunch` → добавить recommendations блок
- `/force_afternoon` → добавить recommendations блок
- `/force_evening` → добавить recommendations + план на завтра
- Проактивні повідомлення (intelligent_listener) → добавить recommendations
